# autoads# 🚗 Auto Ads Bot

Telegram бот для размещения и поиска объявлений о продаже автомобилей с удобным Mini App интерфейсом.

## 📋 Возможности

- 🚘 **Mini App** - удобный каталог с фильтрами и поиском
- 📱 **Адаптивный дизайн** - работает на любых устройствах
- 🔍 **Расширенный поиск** - по марке, модели, цене, году
- 📸 **Загрузка фото** - до 10 фотографий на объявление
- ❤️ **Избранное** - сохраняйте понравившиеся автомобили
- 📊 **Статистика** - просмотры объявлений
- 🔔 **Уведомления** - о новых сообщениях

## 🛠 Технологии

- **Python 3.11** - Telegram бот + API (FastAPI)
- **PostgreSQL 15** - основная база данных
- **Redis** - кеширование и сессии
- **Docker** - контейнеризация
- **HTML/CSS/JS** - Mini App интерфейс

## 🚀 Быстрый старт
